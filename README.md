## Prerequisites
- [Python 3.x](https://www.python.org/)
- [numpy](https://pypi.org/project/numpy/)
    ```sh
    pip install numpy
    ```
- [matplotlib](https://pypi.org/project/matplotlib/)
    ```sh
    pip install matplotlib
    ```
- [keras](https://pypi.org/project/Keras/)
    ```sh
    pip install keras
    ```
